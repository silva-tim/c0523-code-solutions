# command-line-basics-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What is a CLI?
  A command line interface. A way to interact with the computer that doesn't use a regular GUI.

- What is a GUI?
  A graphical user interface. A graphical representation/tool to interact with the computer.

- Give at least one use case for each of the commands listed in this exercise.
  - `man`
    To easily look up manuals/documentation on specific commands.
  - `cat`
    To quickly print the contents of a file to the terminal or to combine two or more files together into one.
  - `ls`
    To easily see what files exist in the current directory.
  - `pwd`
    To print the current directory to the standard output.
  - `echo`
    To print a certain output specified in the command.
  - `touch`
    To easily create a new file.
  - `mkdir`
    To easily create a new directory.
  - `mv`
    To rename or move files or directories.
  - `rm`
    To easily delete files or directories.
  - `cp`
    To easily create a copy of an existing file.
- What are the three virtues of a great programmer?
  Impatience, Laziness, and Hubris.
